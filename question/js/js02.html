var b = a
a.x = a = {n: 2}
console.log(a.n, b.n);
console.log(a.x, b.x);
解析

var b = a; 此时b和a指向同一个对象. b = {n:1}
.运算符比=运算符高，所以先计算a.x
b = {n:1,x:undefinde}
a.x = a = {n:2}; 计算完a.x,再计算 = ，赋值是从右向左的，此时a指向一个新对象
a = {n=2}
再把结果赋值给a.x,
b = {n:1,x:{n=2}}
